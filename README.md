# Gada Solomon — Portfolio

Minimal React + Vite portfolio ready for Vercel deployment.

Setup

```bash
# install
npm install

# dev
npm run dev

# build
npm run build

# preview build
npm run preview
```

Notes

- Uses Bootstrap 5 via CDN in `index.html` for quick styling.
- AOS is initialized in `src/main.jsx`.
- To deploy on Vercel, connect the repository and use the default build `npm run build`.

Files of interest

- `src/components/*` — components for each section
- `src/index.css` — global styling and variables

Customize

Replace `/src/assets/placeholder.svg` and update project links and CV (`/cv.pdf`).
